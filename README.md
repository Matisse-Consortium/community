# Community Code
This repository contains code from members of the MATISSE community or code closely related to it.

> [!NOTE]
> The code provided in this repository might not be entirely well documented.

## Community repositories
* [oimodeler](https://github.com/oimodeler/oimodeler): Interferometric model fitting, by A. Meilland
* [oimodeler-app](https://oimodeler-app.streamlit.app/): Graphical interface for oimodeler by V. Fleury
* [oifits](https://github.com/pboley/oifits): Reading, writing and visualising OIFITS files, by P. Boley
* [p2obt](https://github.com/MBSck/p2obt): Automatic creating observation blocks (OBs) on the ESO P2 interface, by M. B. Scheuck
